# ERC55 Landing Page

This is the reposity for the landing page of ERC55. Made Using the Gatsby Theme [`@lekoarts/gatsby-theme-cara`](https://github.com/LekoArts/gatsby-themes/tree/main/themes/gatsby-theme-cara).


## Setup

```sh
npm install
```

## Develop

```sh
npm run develop
```

## Build

```sh
npm run build
```

## Deploy to Github Pages

```sh
npm run deploy
```
